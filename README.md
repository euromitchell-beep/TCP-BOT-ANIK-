# TCP-BOT-ANIK 

# ⚡ ANIK Control Panel

> Secure Flask + SocketIO Based Admin Control System

---

## 🚀 Overview

ANIK Control Panel is a lightweight web-based admin dashboard built using **Flask** and **Flask-SocketIO**.

It allows:

- 🔐 Secure Admin Login
- ▶ Start / Stop Bot Process
- 📡 Real-Time Log Streaming
- ⏱ Session Handling
- ⚙ Admin Configuration Control

---

## 🛠 Tech Stack

- Python 3
- Flask
- Flask-SocketIO
- Gunicorn
- Eventlet
